길드 타운 사용법

Claude와 Codex 세션을 길드원처럼 보여주는 실시간 현황판입니다. 서버에 연결되지 않으면 안전한 데모 데이터가 표시됩니다.

빠르게 시작하기

  • 길드원 클릭: 상태 확인, 대화창 열기, 터미널 포커스
  • 건물 클릭: 해당 프로젝트에서 새 Claude/Codex 세션 시작
  • 세션 보드: 응답이 필요한 세션을 먼저 확인하고 이름으로 검색
  • 지도 이동: 마우스 휠, 터치 드래그, 또는 키보드 ↑/↓

상태 안내

주황색은 응답 필요, 파란색은 작업 중, 빨간색은 실패 상태입니다. 세션 종료는 확인 후에만 실행됩니다.
